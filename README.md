# Team Member Directory


### Installation

```
npm install
```

### Run project

```
npm run dev
```

### Build project

```
npm run build
```

### Run tests

```
npm run tests
```

Optional: you can change API link in .env file or use default


## About project

- [x] Simple module architecture
- [x] Pinia state manager
- [x] Typescript
- [x] Vue 3 composition API
- [x] Tailwind CSS
- [x] Vite build tool
- [x] Vitest testing library
